# wAIllet DApp

A decentralized application for the wAIllet ecosystem.

## Status

**Under Construction**

This project is being built incrementally. Check back for updates!

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- ethers.js v6

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

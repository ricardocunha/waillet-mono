# wAIllet DApp

A decentralized application for the wAIllet ecosystem.

## Status

**Under Construction**

This project is being built incrementally. Check back for updates!

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- ethers.js v6

## Supported Networks

### Mainnet
- Ethereum
- Base
- BNB Smart Chain

### Testnet
- Sepolia
- Base Sepolia

## Bridge Pairs

| Source | Target |
|--------|--------|
| Ethereum | Base |
| Base | Ethereum |
| Sepolia | Base Sepolia |
| Base Sepolia | Sepolia |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
